# Assignment Log Entry Format

Append one entry per completed assignment. The archived plan preserves the full
reasoning and approach for future reference.

```markdown
## YYYY-MM-DD — Assignment title
**Branch:** agent/<branch-name>
**Commits:** <count> commits, merged to main
**Outcome:** One-line summary of what was delivered.
**Follow-ups:** Any remaining work or related assignments to queue (or "None").

### Archived Plan
<Paste the full contents of plan.md here before clearing it.>
```
