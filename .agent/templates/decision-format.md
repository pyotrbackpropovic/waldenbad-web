# Decision Record Format

File: `.agent/decisions.md`

Append one record per non-trivial architectural or technical choice
(framework, library, schema design, infrastructure pattern, etc).

```markdown
## DR-<number>: <Title>
**Date:** YYYY-MM-DD
**Context:** Why this decision came up.
**Options considered:**
- Option A: Brief description with pros/cons.
- Option B: Brief description with pros/cons.
**Decision:** Which option and why.
**Consequences:** What this means going forward.
```
